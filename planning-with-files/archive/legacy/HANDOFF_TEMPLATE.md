# Handoff Template

Use this exact structure in agent-to-agent handoff notes.

## 1. Scope Completed
- What was completed
- Why it was done

## 2. Files Changed
- List exact file paths

## 3. Validation Run
- Commands executed
- Pass/fail
- Known warnings

## 4. Open Risks
- Technical risks
- Product risks

## 5. Next Actions
1. Immediate next step
2. Follow-up step
3. Optional optimization
