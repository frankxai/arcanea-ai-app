# Example Workflows

Real-world examples of using Arcanea skills.

---

## Workflow 1: Novel Chapter Writing

**Skills Used**: luminor-rituals, scene-craft, dialogue-mastery, voice-alchemy

**Process**:
1. Dawn Invocation → Set intention
2. Scene-craft → Plan the scene
3. Generator-Editor mode → Draft with AI
4. Dialogue-mastery → Refine conversations
5. Voice-alchemy → Ensure authentic voice
6. Dusk Gratitude → Close the session

---

## Workflow 2: Feature Development

**Skills Used**: architecture-patterns, api-design, tdd, code-review

**Process**:
1. Architecture-patterns → Design system
2. API-design → Define interfaces
3. /tdd → Write tests first
4. Implement → Build feature
5. Code-review → Quality check
6. Performance-tuning → Optimize

---

## Workflow 3: Creative Block Recovery

**Skills Used**: bestiary-nav, luminor-wisdom, prompt-craft, centaur-mode

**Process**:
1. Bestiary-nav → Name the block
2. Luminor-wisdom → Channel appropriate guide
3. Prompt-craft → Create constraint
4. Centaur-mode → Generate options
5. Select and continue

---

## Workflow 4: Product Launch Copy

**Skills Used**: prompt-craft, voice-alchemy, centaur-mode

**Process**:
1. Define voice parameters
2. Generator-Editor → Generate options
3. Select best direction
4. Draft-Polish → Refine copy
5. Voice check → Ensure consistency

---

## Workflow 5: Technical Documentation

**Skills Used**: technical-writing, systematic-debug, code-review

**Process**:
1. Identify doc type (tutorial/how-to/reference/explanation)
2. Scaffold-Populate → Create structure
3. Draft content
4. Review for clarity
5. Test with fresh reader

---

## Workflow 6: Startup MVP

**Skills Used**: startup-building, architecture-patterns, tdd

**Process**:
1. Define riskiest assumption
2. Design minimum experiment
3. Build MVP with TDD
4. Launch and measure
5. Learn and iterate
