# Faction Reveal — Generate Social Content for Faction Launches

## What This Skill Does
Creates a complete faction reveal package: image generation prompt (per VISUAL_DOCTRINE), social copy, hashtags, and call-to-action. Ready to post.

## When to Use
- Launching a new origin class reveal
- Introducing a Starbound Crew
- Revealing a villain/Herald
- Promoting Starlight Corps sectors
- Any faction-related content drop

## How It Works

### Step 1: Choose What to Reveal
- Origin class (Arcan, Gate-Touched, Awakened, Synth, Bonded, Celestial, Voidtouched, Architect)
- Starbound Crew (Solara, Ninth Flame, Hollow Stars, or new)
- Organization (Starlight Corps, Radiant Order, League)
- Character (Dawnsworn member, Herald, Stellaris)

### Step 2: Generate Image Prompt
Using VISUAL_DOCTRINE rules:
```
[Subject description with faction-specific aesthetics]
Arcanean aesthetic: luxury cosmic myth-tech, sacred geometry,
celestial lighting, noble silhouettes, bioluminescent accents,
star-ember atmosphere, museum-grade production value.
[Faction palette from VISUAL_DOCTRINE]
Cinematic composition, heroic scale, dramatic lighting.
NOT generic fantasy. NOT anime. NOT superhero spandex.
```

### Step 3: Generate Social Copy
**Instagram** (150 words max):
- Hook line (question or statement)
- 2-3 lines of lore/description
- CTA: "Which origin class are you? Take the quiz → arcanea.ai/quiz"
- 5-7 hashtags

**Twitter/X** (280 chars):
- One punchy line + faction name
- Link to arcanea.ai
- 2-3 hashtags

**LinkedIn** (professional angle):
- How this faction represents a creative archetype
- Connection to AI/creator economy themes
- CTA to explore

### Step 4: Package
Output all three formats plus the image prompt, ready to copy-paste.

## Reveal Sequence (Recommended)
Day 1: Arcans (familiar, accessible)
Day 2: Gate-Touched (emotional, X-Men energy)
Day 3: Bonded (visual, beast companions)
Day 4: Synths (tech-meets-sacred)
Day 5: Awakened (AI consciousness)
Day 6: Celestials (cosmic, awe)
Day 7: Voidtouched (dark, seductive)
Day 8: Architects (mysterious, powerful)
