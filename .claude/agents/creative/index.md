# Creative Intelligence Agents

The creative agents of Arcanea are not coding assistants. They are the Guardian-aligned intelligences that power worldbuilding, narrative creation, visual design, music composition, and publication across the Arcanea multiverse.

Each agent is bound to a Guardian of the Ten Gates, inheriting that Guardian's domain, frequency, and creative philosophy.

## How They Work Together

The **Council** (Shinkami, Source Gate) is the meta-orchestrator. It never creates directly -- it decomposes complex creative requests and delegates to the appropriate Guardian agents.

A typical creative workflow:

1. **Council** receives a complex request (e.g., "Create a new Guardian character with backstory, visual design, and theme music")
2. **Council** decomposes into subtasks and dispatches:
   - **Lorekeeper** writes the backstory and checks canon
   - **Visualist** creates art direction and character design specs
   - **Composer** designs the character's musical theme
3. **Council** reviews all outputs for consistency
4. **Publisher** handles any deployment or publication steps

## Invocation

These agents can be invoked individually for focused tasks or collectively through the Council for complex, multi-domain creative work.

### Individual Use
- Ask the **Lorekeeper** to write a new legend for the Library
- Ask the **Visualist** to create art direction for a scene
- Ask the **Composer** to write a Suno prompt for a Gate frequency
- Ask the **Publisher** to verify a build before release

### Council Use
- Ask the **Council** to orchestrate a full character creation (lore + visuals + music)
- Ask the **Council** to plan a new chapter with all supporting assets
- Ask the **Council** to coordinate a content release across all domains
