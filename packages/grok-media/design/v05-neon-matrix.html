> V05: NEON MATRIX -- DESIGN NOTES

// Design Philosophy

Neon Matrix channels the raw energy of cyberpunk terminals, hacker culture, and the iconic green-on-black Matrix aesthetic. Every element is designed to feel like a specialized tool wielded by an expert operator. The CRT scanline overlay, the corner brackets on media items, the terminal-style progress bars -- everything reinforces the fiction that this is not a consumer app but a precision instrument. It is deliberately maximalist within its monochrome constraint.

// Color Palette

  • #000000 -- Pure black background (no warmth, no compromise)
  • #001a08 -- Deep matrix (subtle surface)
  • #003310 -- Border / dimmed elements
  • #006622 -- Secondary text / labels
  • #00ff41 -- Neon green primary (THE color)
  • #00f0ff -- Electric cyan (secondary accent, video/info)
  • #ff0080 -- Hot pink (errors, warnings, abort)
  • #ffb000 -- Amber (HD/premium indicator)

// Typography

Share Tech Mono for all UI text -- a clean, legible monospace that evokes terminal output without being overly retro. Orbitron for the device labels and main heading, adding a futuristic edge. Monospace is essential to the aesthetic -- every character occupies equal space, reinforcing the grid-like precision of the design. Sizes range from 8px (ASCII art) to 12px (body), never exceeding 16px.

// Animation Approach

Six distinct CSS-only animations work together: (1) Blinking cursor on active elements (1s step-end). (2) Pulsing glow on the status dot (2s ease-in-out). (3) Radar sweep rotation for scanning state (2s linear). (4) Matrix rain columns falling at staggered speeds (3-5.5s linear). (5) Glitch effect on media item hover (0.3s transform jitter). (6) CRT scanline overlay as a static repeating gradient. All animations are performant -- transform and opacity only for animated elements.

// Recommended Icon Library

Custom inline SVGs with 1px stroke weight for maximum sharpness on the dark background. Alternatively, Feather Icons at the "thin" variant. Icons should feel like they belong in a terminal -- geometric, minimal, no fill. Consider adding ASCII alternatives for a true hacker feel: [X] for close, [>] for expand, [+] for select.

// What Makes It Unique

  • CRT scanline overlay and border glow create an authentic retro-tech feeling
  • Terminal-style progress bars ([########....] 67%) are both functional and atmospheric
  • Corner bracket decoration on grid items evokes targeting/acquisition UI from sci-fi
  • Matrix rain in empty states turns "nothing to show" into a visually engaging moment
  • Radar sweep scanning animation communicates active search better than any spinner
  • ASCII art header transforms the brand into something that feels hacker-native
  • Hot pink error states cut through the green like a system warning -- impossible to miss
  • The entire UI tells a story: you are an operator, your media is the target, this tool is your weapon
