V04: Midnight Minimal — Design Notes

Design Philosophy

Midnight Minimal is an exercise in restraint. Inspired by Linear's information density, Vercel's monochrome confidence, and Raycast's utility-first elegance. Every pixel serves a purpose. No decoration, no ornament — just information, space, and the occasional flash of teal to guide the eye. The design proves that dark UI can feel light when you trust whitespace.

Color Palette

  • #09090B — Background (near-black, warm undertone)
  • #18181B — Surface / Elevated
  • #27272A — Border (barely visible)
  • #3f3f46 — Border hover
  • #52525B — Muted text
  • #A1A1AA — Secondary text
  • #FAFAFA — Primary text
  • #7fffd4 — Crystal Teal (single accent, used sparingly)

Typography

Inter with -0.02em letter-spacing throughout. Weights: 300 (light emphasis), 400 (body), 500 (labels), 600 (headings). Size scale: 9px (shortcut hints), 10px (footer), 11px (metadata/filters), 12px (body), 13px (titles). No size larger than 13px — density over drama.

Animation Approach

Minimal transitions only. 150ms ease on hovers (background and color shifts). 200ms for toggle switches. 300ms for progress bar fills. The scanning state uses a simple back-and-forth pulse on the progress bar. A single rotating icon for the scan spinner. Nothing bounces, nothing glows, nothing demands attention.

Recommended Icon Library

Lucide Icons or Radix Icons — 1.5px stroke weight, matching the hairline border aesthetic. All icons should be 14px in buttons and 12px inline. Phosphor would also work at the "thin" weight.

What Makes It Unique

  • 2px media grid gap creates a dense, intentional mosaic that lets content breathe through color alone
  • Keyboard shortcut hints (S, D, 1/2/3) are visible but subordinate — power-user friendly without cluttering
  • Selection uses a teal corner marker with checkmark, not a glow or overlay — clean and decisive
  • Ghost buttons by default, solid teal only for the primary action — clear hierarchy
  • The entire design uses exactly one accent color, proving that constraint breeds elegance
  • Every element aligns to a strict 4px grid — nothing is arbitrary
