V03 -- Aurora Flow

Design Philosophy

Inspired by Runway ML's ethereal gradients and Midjourney's dark, organic interfaces. This variation embraces flowing movement — aurora gradients that shift through teal, blue, purple, and pink, giving every element a sense of living, breathing energy. The deep navy background (#0B1120) is warmer than pure black, creating a sense of depth that pure dark themes often lack. Every interaction has a flowing, organic quality rather than mechanical precision.

Color Palette

  • #0B1120 — Deep navy background
  • #5ee7c8 — Aurora teal (primary accent)
  • #4a9eff — Aurora blue
  • #9b6dff — Aurora purple
  • #ff6db5 — Aurora pink
  • #e8eaf4 — Primary text
  • #4a4e68 — Secondary text
  • #b0b4cc — Body text

Typography

  • DM Sans — Modern geometric sans-serif, softer than Inter
  • Weights: 300 (light labels), 400 (body), 500 (UI), 600 (emphasis), 700 (headings)
  • Generous letter-spacing on smaller sizes for readability
  • Gradient text via background-clip for percentage displays

Animation Approach

  • Aurora border sweep — 4-color gradient rotating around the panel edge via @property
  • Wave header bar — Top bar with flowing 4-color gradient shifting position
  • Blob decorations — 3 soft blobs floating in the background with offset timing
  • Aurora tab underline — Gradient underline that both slides and color-shifts
  • Card hover aurora ring — On hover, media cards get a full aurora gradient border with hue-rotate
  • Triple-ring scanner — 3 concentric rings with different speeds and colors
  • Breathing orb — Empty state orb scales and pulses with expanding ring waves
  • Progress bar aurora flow — Download bars have flowing gradient + glow dot at leading edge
  • FAB ripple expansion — Radial gradient pseudo-element scales up on hover

Recommended Icon Library

  • Inline SVG with round linecaps, 2px stroke weight
  • If external: Lucide Icons — clean, consistent, round aesthetic
  • Icons should feel soft and organic, matching the flowing design language

What Makes It Unique

  • Four-color aurora system — Unlike V01's two-color scheme, this uses teal, blue, purple, and pink in flowing gradients that shift and breathe across every element
  • Organic blob decoration — Background blobs drift with offset timing, creating a living, generative feel
  • Deep navy warmth — #0B1120 instead of pure black gives the entire UI a warmer, more inviting atmosphere
  • Hue-rotating card borders — On hover, card aurora borders shift through the color spectrum via CSS hue-rotate
  • Triple-ring scanner — Three concentric rings at different speeds and directions create a complex, mesmerizing loading state
  • DM Sans typography — Geometric but soft, differentiating from V01's Inter and V02's monospace
  • Aurora-gradient scrollbar — Even the scrollbar uses the four-color gradient, maintaining the theme to the last pixel
  • Toggle with glow shadow — Active toggles emit a teal-to-blue glow, reinforcing the aurora brand
