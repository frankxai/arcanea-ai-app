V01 — Crystal Glass Evolution

Design Philosophy

Evolution of Arcanea's existing glassmorphism, pushed to Apple Vision Pro levels of depth and refinement. Every element uses layered transparency, animated gradients, and micro-interactions that make the extension feel like a premium native app rather than a web panel. The design prioritizes information density without visual clutter through careful use of opacity layers and subtle borders.

Color Palette

  • #0a0515 — Deep void background
  • #0f0a1e — Surface layer
  • #8b5cf6 — Violet primary
  • #7fffd4 — Crystal teal accent
  • #ffd700 — Gold for HD badges
  • #e2e0ea — Primary text
  • #8a8599 — Secondary text
  • #5a5470 — Muted text

Typography

  • Inter — All UI text, weights 300-700
  • Tabular numerals for all counters and sizes
  • Brand name uses gradient text via background-clip
  • Scale: 9px (badges) / 11px (labels) / 12px (body) / 13px (settings) / 14px (headers)

Animation Approach

  • @property animations — CSS Houdini for gradient rotation on panel borders
  • Cubic bezier easing — 0.4, 0, 0.2, 1 for natural motion curves
  • Shimmer loading — Background-position animation on skeleton elements
  • Progress bar particles — Pseudo-element translateX for shimmer effect
  • Tab sliding indicator — Width + left transition with spring-like timing
  • Selection checkmark pop — Scale bounce on selection
  • FAB pulse ring — Scale + opacity pulse for call-to-action glow
  • Dual-ring scanner — Nested counter-rotating borders
  • Floating orb — Vertical translate for empty state

Recommended Icon Library

  • Inline SVG (as implemented) for zero-dependency operation
  • If external library needed: Phosphor Icons (matches Arcanea's existing system)
  • Stroke-based, 2px weight, round caps

What Makes It Unique

  • Animated conic gradient border — The panel edge has a rotating sweep of teal-to-violet that is subtle but unmistakably premium
  • Layered glass depth — Three distinct depth levels: background, card surface, and overlay, each with appropriate transparency
  • Masonry grid support — Tall items span 2 rows via grid-row: span 2 for visual variety
  • Crystal-themed scrollbar — Custom scrollbar with teal tint matching the glass theme
  • Selection glow animation — Selected items emit a brief inward glow pulse on selection
  • Action bar slide — Selection bar animates up from below with spring motion
