V02 -- Neobrutalist Edge

Design Philosophy

Anti-design as intentional design. This variation rejects the smooth, blurred, gradient-heavy aesthetic dominating 2025-2026 UI and replaces it with raw, unapologetic structural clarity. Every pixel serves a purpose. Hard shadows create physical depth. Monospace typography turns the interface into a command terminal with visual power. The constraint of ONE accent color (crystal teal) forces every element to earn its emphasis. Inspired by brutalist architecture — concrete, exposed structure, honest materials.

Color Palette

  • #000000 — Pure black background
  • #FFFFFF — Structural white
  • #7fffd4 — Crystal teal (THE accent)
  • #222222 — Grid lines, borders
  • #444444 — Inactive elements
  • #888888 — Body text

Typography

  • JetBrains Mono — 100% monospace, all elements
  • All headers uppercase with wide letter-spacing (2-4px)
  • Weights: 400 (body), 600 (labels), 700 (emphasis), 800 (headers/counters)
  • ASCII art progress bars: [||||||||------] 80%
  • Large bold overlay numbers on media cards (#001, #002...)

Animation Approach

  • Minimal and mechanical — 100-150ms transitions, ease timing only
  • Hard shadow shifts — Buttons translate on hover, shadows extend
  • Step-based rotation — Scanner uses steps(4) for mechanical feel
  • Color inversion on hover — Scan button inverts fill/stroke
  • No gradients, no blur — Flat color transitions only
  • Pulse is binary — Scanner inner square toggles opacity 1/0.2

Recommended Icon Library

  • Inline SVG with square linecaps (stroke-linecap="square")
  • If external: Tabler Icons at 2.5px stroke weight
  • All icons should feel geometric and constructed

What Makes It Unique

  • ASCII progress bars — Download progress shown as [||||----] text blocks, referencing terminal culture
  • Hard shadow system — 8px 8px 0 offset shadows on panels, 4px on buttons, creating physical depth
  • Bold numbering — Every media item has a large overlay number, making the grid feel like a contact sheet
  • Selection counter — Selected items show their selection order (01, 02, 03) rather than just a checkmark
  • Zero border-radius — Everything is sharp-cornered except full-round elements, enforcing the brutalist ethos
  • Visible grid structure — Grid lines, page counters, and structural labels are visible, not hidden
  • One-color discipline — The restriction to #7fffd4 as the ONLY color creates powerful visual hierarchy
