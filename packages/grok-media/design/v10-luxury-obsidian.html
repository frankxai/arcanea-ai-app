Design Notes

Variation X — Luxury Obsidian

Philosophy

Luxury Obsidian draws from the visual language of high-end fashion houses and automotive brands — Cartier, Bottega Veneta, Porsche Design. The principle is restraint as luxury: every pixel of whitespace is intentional, every border is a hairline, every transition is luxuriously slow. The obsidian-black foundation creates an infinite depth that makes the warm gold accents feel like precious metal emerging from darkness.

The design avoids the common trap of "dark mode = lots of borders and shadows." Instead, it uses the absence of visual noise as its primary luxury signal. Content floats on near-invisible surfaces. Gold is used with surgical precision — only for active states, brand identity, and interactive affordances. The carbon fiber texture in the header is a nod to Porsche Design's material language, grounding the digital experience in physical craft.

Typography pairs Cormorant Garamond (an elegant serif with beautiful numerals) with Outfit (a clean geometric sans-serif). The contrast between ornate display type and minimal UI text creates the same tension found in luxury brand communications — heritage meets modernity.

Typography

  • Display / Headers: Cormorant Garamond (300, 400, 500) — Elegant serif with beautiful numeral forms. Used for stat numbers, section titles, brand name fallback, and the scanning counter.
  • Body / UI: Outfit (200, 300, 400, 500) — Clean geometric sans-serif. Used for labels, descriptions, buttons, and all interactive text.
  • Letter Spacing: 0.15em (wide), 0.20em (wider), 0.25em (widest) — Generous spacing on all uppercase text, a hallmark of luxury print design.
  • Weight Range: Predominantly 200-300 (ultralight to light). Weight 500 reserved exclusively for active/selected states. No bold anywhere — luxury whispers.

Key Design Elements

  • Borders: 0.5px gold-tinted hairlines. Primary borders use rgba(201,169,96,0.18). Subtle borders use rgba(255,255,255,0.04). Selection borders use rgba(201,169,96,0.60) — precise, not glowing.
  • L-Shaped Corner Accents: Cards feature pseudo-element gold corners (top-left, bottom-right) that intensify on hover. A signature detail borrowed from luxury packaging.
  • Carbon Fiber Texture: Header background uses repeating-linear-gradient in both axes (2px/4px grid) at 1-1.5% white opacity. Subtle enough to feel tactile, not decorative.
  • Grain Overlay: Full-viewport SVG feTurbulence noise at 3.5% opacity with mix-blend-mode: overlay. Adds photographic texture to the digital surface.
  • Gold Gradient Text: Brand name uses background-clip: text with a 200% oversized gradient that animates position over 6 seconds for a metallic shimmer effect.
  • Gold Hairline Dividers: Section separators use linear-gradient(90deg, transparent, gold, transparent) for elegant fade at edges.
  • Tab Bar: Zero visual indicator (no underline, no background). Active state is communicated purely through gold color + weight 500. Maximum restraint.
  • Media Grid: Uniform 1:1 aspect ratio with minimal 4px gaps. Selection is a precise gold border, not a glow or overlay. Check mark appears on a solid gold square.
  • Hover Effects: Extremely subtle — 2% lighter background, faint gold border fade-in. The popup action buttons use a left-to-right gold wash fill on hover.
  • Progress Bars: 1px thin gold lines with a 3px glowing dot at the leading edge. Active downloads pulse the dot. Percentage shown in large Cormorant Garamond numerals.
  • Toggles: Minimalist 32×16px with a 10px circle. Off: gray on dark. On: gold circle on gold-tinted background with gold border.
  • Empty State: Gold line art (0.5px stroke SVG) of a media frame with landscape composition lines. No fill, no color — pure draftsmanship.
  • Stats: Large Cormorant Garamond numbers (26px side panel, 20px popup) with tiny 8px uppercase Outfit labels below. Selected count uses gold.

Animations

  • Transition Timing: All interactions use 400ms cubic-bezier(0.4, 0, 0.2, 1) — luxuriously slow, never snappy. The ease curve decelerates gently, like a luxury car braking.
  • Gold Shimmer: Brand name gradient animates background-position over 6s with the same cubic-bezier. Creates a slow metallic shimmer.
  • Scanning Rings: Dual counter-rotating rings (3s / 2s) with a pulsing center dot (1.5s). The outer ring has a gold top-border, inner has a gold bottom-border, creating orbital motion.
  • Scan Progress: A 40%-width gold gradient bar slides across the track in 2s, creating an indeterminate progress feel.
  • Stagger Reveal: Grid items animate in with 50ms stagger (0.6s duration each), translating up 8px from transparent. Pure CSS, no JS.
  • Progress Pulse: Active download leading dots pulse opacity between 0.7 and 1.0 over 1.5s.
  • Breathing Glow: During scanning, the panel border subtly pulses with a 20px gold box-shadow over 3 seconds.

UI States Shown

  • Side Panel — Library Grid: 16 media items (4 selected) with badges, filter bar, selection bar with export button, stats overview
  • Side Panel — Downloads: 2 active downloads with progress bars and percentages, 2 completed downloads
  • Side Panel — Settings: 3 setting groups (Export, Quality, Storage) with toggles and selects
  • Side Panel — Scanning: Dual-ring animation, media counter, indeterminate progress bar
  • Side Panel — Empty: Gold line art, descriptive text, scan CTA button
  • Popup — Main: Stats, 3 quick actions, 6-item recent grid, footer
  • Popup — Scanning: Compact scanning state with reduced ring size
  • Popup — Downloads: Compact download list with progress percentages
  • Popup — Settings: Simplified 4-row settings with compact toggles
  • Popup — Empty: Minimal empty state with line art and message

Inline SVG Icons

  • Scan: Viewfinder frame (4 L-corners) with center circle — used in header, scan button, CTA
  • Download: Down arrow into tray — used in export button, popup action
  • Settings: Gear with center circle — used in header, popup header
  • Filter: Three horizontal lines descending in width — used in header
  • Video Badge: "VID" text badge with rose border — used on video media items
  • HD Badge: "HD" / "4K" text badge with gold border — used on high-res items
  • Checkmark: Polyline checkmark on solid gold background — used for selection state
