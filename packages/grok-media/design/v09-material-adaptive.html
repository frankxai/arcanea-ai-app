Design Notes

Variation 09 — Material Adaptive applies Google's Material Design 3 (Material You) design language to the Arcanea Media extension, creating a product that feels native to the Chrome ecosystem while retaining Arcanea's identity through color adaptation.

Design Philosophy

Material Design 3 introduces the concept of dynamic color derived from content. Here, the primary purple (#D0BCFF) provides brand identity while the adapted teal secondary (#7fffd4) anchors the Arcanea palette. Surface tones use elevation tinting rather than shadows — each level adds a subtle wash of primary color over the dark surface, creating depth without hard edges.

Every interactive element follows M3 state layer conventions: hover overlays at 8% opacity, press at 12%, focus at 12% with visible outline. The result is a design that breathes, responds, and feels native to Google's ecosystem while carrying Arcanea's creative identity.

M3 Component Inventory

This design implements the following Material Design 3 components:

Navigation: Top App Bar (small), Segmented Button Group (3-segment, replaces underline tabs per M3 guidelines)

Selection: Outlined Filter Chips (checkmark on active state), Checkbox overlays on media grid items

Actions: Extended FAB (icon + label, primary-container fill), Small FAB, Filled Button, Outlined Button, Icon Button (48dp touch target with 40dp visible area)

Feedback: Linear Progress Indicator (4dp, indeterminate animation), Snackbar (with action button), Dual-ring scanning spinner

Layout: Outlined Cards (1dp border, no shadow), Surface-tinted elevation (5 levels), Content list items (56dp minimum height), Stat cards

Controls: Toggle Switch with thumb icon indicators — checkmark SVG for on, X SVG for off, with M3-correct thumb scaling from 16dp to 24dp

Animation Details

Grid stagger: Each media item enters with scale(0.92) translateY(8px) to scale(1) translateY(0) using emphasized deceleration easing with 50ms stagger delay. Total cascade: 750ms delay + 400ms duration = 1.15s for all 16 items.

Indeterminate progress: Track slides translateX(-100%) to translateX(100%) with scale oscillation (0.3 to 0.6 and back), 2s period, emphasized easing for the characteristic M3 "rubber band" feel.

Snackbar entry: translateY(20px) with emphasized deceleration, 300ms duration. Matches Android Material snackbar behavior.

Switch toggle: Thumb slides with emphasized easing while growing from 16dp to 24dp. Icon crossfades via fill-color transition from transparent to the primary color.

Accessibility

All interactive elements maintain 48dp touch targets per M3 specification. Color contrast ratios exceed WCAG AA: on-surface text on surface achieves 13.5:1, on-surface-variant text achieves 7.8:1. Selected states communicate through both outline color and checkmark icon, never color alone. Focus states use 12% state layer with visible ring. Segmented buttons and icon buttons include aria-label attributes.

Inline SVG Icon Reference

All icons are Material Symbols (filled variant), embedded as inline SVGs for zero-dependency rendering:

search (scan), download (download/export), settings (gear), filter_list (filter), play_arrow (video badge), hd (HD badge), check (checkmark/selected), close (X / switch off), more_vert (overflow menu), image (empty state), folder_open (save location), delete (clear cache), notifications (alerts), shield (connections), code (filename pattern)
