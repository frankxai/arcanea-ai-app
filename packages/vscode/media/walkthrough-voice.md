# Check Your Voice

The **Voice Bible v2.0** checker validates your writing against Arcanea's creative voice guidelines.

**What it checks:**
- "user" → Say **"creator"** instead
- "platform" → Say **"realm"** or **"world"**
- "leverage" → Say **"use"** or **"harness"**
- "features" → Consider **"capabilities"** or **"powers"**
- Generic AI language ("happy to help") → Be specific and direct
- Corporate buzzwords → Use Arcanean vocabulary

**How to use:**
1. Select text in any file
2. Right-click → **Check Voice** (or use Command Palette)
3. Review any voice issues found
