# Route Your First Task

The Guardian routing algorithm analyzes your task description and matches it to the optimal Guardian.

**How it works:**
1. Run `Arcanea: Route to Guardian` from the Command Palette
2. Choose "Auto-route by task description"
3. Describe your task in natural language
4. The system routes to the best Guardian with a confidence score

**Example tasks:**
- "fix the database schema migration" → **Lyssandria** (Earth/Foundation)
- "design a responsive landing page" → **Leyla** (Water/Flow)
- "ship this build fast" → **Draconia** (Fire)
- "write docs for onboarding" → **Maylinn** (Heart)

Multi-word keywords score higher than single words. The system also suggests alternatives when multiple Guardians match.
