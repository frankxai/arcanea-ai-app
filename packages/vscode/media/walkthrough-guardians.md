# Meet Your Guardians

The **Guardian Panel** in the Arcanea sidebar shows all 10 Guardians of the Ten Gates.

Each Guardian specializes in a creative domain:

| Guardian | Domain |
|----------|--------|
| **Lyssandria** | Infrastructure, database, security |
| **Leyla** | Design, UI/UX, animations |
| **Draconia** | Execution, performance, shipping |
| **Maylinn** | Documentation, community, empathy |
| **Alera** | Brand expression, naming, messaging |
| **Lyria** | Strategy, AI/ML, pattern recognition |
| **Aiyami** | Wisdom, mentorship, knowledge |
| **Elara** | Perspective, reframing, innovation |
| **Ino** | Integration, APIs, collaboration |
| **Shinkami** | Meta-architecture, orchestration |

Click any Guardian in the sidebar to activate them. The status bar shows your active Guardian.
