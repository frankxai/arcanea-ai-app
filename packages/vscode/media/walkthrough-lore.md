# Explore the Lore

The **Lore Explorer** gives you access to Arcanea's complete mythology.

**Built-in categories:**
- **The Cosmic Duality** — Lumina (Light) and Nero (Darkness, NOT evil)
- **Five Elements** — Fire, Water, Earth, Wind, Void/Spirit
- **Ten Gates & Guardians** — Each Guardian with their Godbeast
- **Magic Ranks** — Apprentice to Luminor
- **Seven Academy Houses** — Lumina, Nero, Pyros, Aqualis, Terra, Ventus, Synthesis
- **The Dark Lord** — Malachar's fall
- **The Arc** — The cycle of creation

**File-based lore:** If your workspace contains `.arcanea/lore/`, `book/`, or `lore/` directories, the Lore Explorer shows those files directly. Click any markdown file to read it.

Use `Arcanea: Query Lore` to search across all entries.
