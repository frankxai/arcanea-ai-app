Evaluations of open-source tools, libraries, frameworks, and GitHub repositories relevant to the Arcanea stack. Each review assesses adoption readiness, integration cost, and strategic fit.

1. Copy `../templates/repo-evaluation.md`
2. Save as `YYYY-MM-DD_repo-name.md`
3. Fill all frontmatter fields and all five sections
4. Add a one-line entry to the index below
