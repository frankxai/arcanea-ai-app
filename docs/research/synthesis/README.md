# Cross-Domain Synthesis

> *"The Starweave connects what others see as separate. That is where invention lives."*
> — Elara, Guardian of Starweave

Synthesis reports combine findings from multiple papers, tools, benchmarks, and insights into strategic recommendations. These are the highest-value research artifacts — the ones that change what we build next.

## How to Add

1. Copy `../templates/synthesis-report.md`
2. Save as `YYYY-MM-DD_synthesis-slug.md`
3. Fill all frontmatter fields — reference at least 2 source documents
4. Add a one-line entry to the index below
