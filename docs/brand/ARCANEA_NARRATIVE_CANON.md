# Arcanea Narrative Canon

Last updated: 2026-03-27

## Core Positioning

Arcanea is a creator platform for world-builders and ambitious creators.

It helps people:

- chat with AI partners that think in systems
- build coherent worlds, stories, and creative artifacts
- learn from a deep philosophy and lore library
- grow from one-off output into a durable creative practice

## Public Message Hierarchy

### What it is

Arcanea is creative intelligence for world-builders.

### What it does

It combines chat, creation tools, and a philosophy library in one system.

### Why it is different

It is not just a generic chat UI or prompt wrapper. It has a stronger point of view, deeper continuity, and a richer creative framework than commodity AI tools.

## Voice Rule

Public acquisition copy should be plain-language first and mythic second.

Use mythology to:

- deepen memorability
- explain the philosophy layer
- reward users who go deeper

Do not use mythology to:

- obscure basic product understanding
- replace clear CTAs
- make pricing, onboarding, or README copy harder to parse

## Approved Short Blurfs

### 1-line

Arcanea is creative intelligence for world-builders and ambitious creators.

### 2-line

Chat with AI, build worlds, and create stories, visuals, and ideas in one coherent system. Arcanea combines creator tools with a deep philosophy library that gives the work more structure and depth.

### README / About

Arcanea is a creator platform that helps people think, build, and create with AI. The product combines chat, world-building, creative tooling, and a deep philosophy layer so creators can move from scattered ideas to coherent output.

## Hero Formula

Homepage hero should always answer four questions in order:

1. Who is this for?
2. What can I do here right now?
3. Why is this different from a generic AI tool?
4. What should I click first?

Recommended structure:

- Headline: creator outcome
- Subheadline: chat + create + world-build
- Trust line: philosophy depth, specialist intelligences, free-to-start
- Primary CTA: create/chat
- Secondary CTA: learn more or explore library

## GitHub Repo Description Formula

Use this pattern:

`[Repo name] is the [specific role] in the Arcanea creator ecosystem.`

Examples:

- `Arcanea is the main creator platform and ecosystem control plane.`
- `@arcanea/mcp-server connects Arcanea capabilities to MCP-compatible clients.`
- `author-os-cli is a workflow tool for AI-native long-form production.`

## Mythic Vocabulary Guidance

Keep:

- Guardians
- Luminors
- Gates
- Lore
- Philosophy

Translate on first mention:

- Guardians = specialist creative intelligences
- Luminors = AI companions or AI partners
- Gates = creative disciplines or modes

Avoid as public front-door copy:

- creative multiverse
- open-source civilization
- superintelligence that makes every model feel like magic
- six layers, one ecosystem

These can still appear deeper in lore, brand, and world pages.
