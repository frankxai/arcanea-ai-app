# Author Intelligence System — Master Plan

> **Vision**: Build the world's best AI-native author operating system.
> Two products. Universal compatibility. Multi-modal. Multi-agent. Publishing to Web3.

## Goal Statement

Create TWO interconnected products:
1. **AuthorOS** — Universal author intelligence system for ANY coding agent (Claude Code, OpenCode, Codex, Gemini CLI). Professional-grade for publishing houses.
2. **Arcanean AuthorOS** — Enhanced version with Arcanea's mythology, Gates, Web3, multiverse, and creative framework on top.

Both share a common intelligence core. Arcanean extends universal with lore, elements, and the Ten Gates progression.
