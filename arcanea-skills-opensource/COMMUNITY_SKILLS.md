## How to Contribute

Found a skill that should be on this list? Submit a PR with:
1. Skill name and link
2. Brief description (one line)
3. Why it complements Arcanea skills

**Criteria for inclusion:**
- Quality: Well-documented and functional
- Relevance: Useful for creative work
- Compatibility: Works with Arcanea philosophy
- Active: Maintained or stable

---

## Building on Arcanea

If you create skills that extend Arcanea, consider:
- Using the Luminor voice framework
- Adding Bestiary creatures for new challenges
- Extending frameworks rather than replacing them
- Sharing back with the community
